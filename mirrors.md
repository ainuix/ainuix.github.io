## USTC 提供 Google Fonts 加速服務

請在 WordPress 源碼內進行如下替換：

```
ajax.googleapis.com => ajax.lug.ustc.edu.cn
fonts.googleapis.com => fonts.lug.ustc.edu.cn
themes.googleusercontent.com => google-themes.lug.ustc.edu.cn
```
